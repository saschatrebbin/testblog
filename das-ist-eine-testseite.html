Willkommen auf dieser Testseite!

Diese Webseite wurde erstellt, um eine einfache und effiziente Möglichkeit zu testen, wie man statische Webseiten mit modernen Tools wie Publii, GitHub und Cloudflare Pages erstellen und hosten kann.

Hier ist der Ablauf, den ich verwendet habe:

  1. Erstellung der Webseite mit Publii: Publii ist ein Desktop-Webseitenbaukasten, der es ermöglicht, statische Webseiten ohne Programmierkenntnisse zu erstellen. Es bietet eine benutzerfreundliche Oberfläche und erlaubt es, Inhalte wie Blogposts, Seiten und Menüs einfach zu verwalten.

  2. Versionierung mit GitHub: Die erstellte Webseite wird in einem GitHub-Repository gespeichert. Dies bietet den Vorteil der Versionskontrolle, sodass Änderungen nachverfolgt und bei Bedarf rückgängig gemacht werden können. Außerdem dient GitHub als Backup-Lösung.

  3. Bereitstellung über Cloudflare Pages: Cloudflare Pages ist ein kostenloser Hosting-Dienst für statische Webseiten. Es integriert sich nahtlos mit GitHub und stellt die Webseite automatisch bereit, sobald Änderungen im Repository vorgenommen werden. Cloudflare bietet auch kostenlose SSL-Zertifikate, sodass die Webseite sicher über HTTPS erreichbar ist.

Das Ziel dieses Tests ist es, eine einfache, wartungsarme Lösung zu finden, die meinem Freund hilft, sein Portfolio für sein Nebengewerbe zu präsentieren. Wenn du mehr über diese Tools erfahren möchtest, schau dir die jeweiligen Webseiten an:

  • Publii
  • GitHub
  • Cloudflare Pages
